Business Intelligence data request:

I am looking for the normal/abnormal results for 'SN shot' cPL and fPL. Columns: SAP, SN, kit type, lot number, result (normal/abnormal)
The date range for this request is:
most recent completed 12 months.
SQL query:
SELECT
DW.SDX_DATA_TYPE_7.LOT_STRING,
CASE
WHEN DW.SDX_DATA_TYPE_23.RESULT = 1 THEN 'Normal'
ELSE 'Abnormal'
END AS RESULT,
CASE
WHEN DW.SDX_DATA_RUN.TEST_TYPE = 5 THEN 'cPL'
ELSE 'fPL'
END AS KIT_TYPE,
DW.BASIC_CUSTOMER_CURR.SAP_NUMBER_CONVERSION
DW.IVLS_INSTALLATION.IVLS_SERIAL_NUMBER
FROM
DW.CUSTOMER_CURRENT BASIC_CUSTOMER_CURR,
DW.SDX_DATA_TYPE_7,
DW.SDX_DATA_RUN,
DW.SDX_DATA_TYPE_23,
DW.IVLS_INSTALLATION,
DW.SDX_DATA_RESULT
WHERE
( IVLS_INSTALLATION.IVLS_INSTALLATION_ID=SDX_DATA_RUN.IVLS_INSTALLATION_ID )
AND ( BASIC_CUSTOMER_CURR.CUSTOMER_CURRENT_ID=SDX_DATA_RUN.CUSTOMER_CURRENT_ID )
AND ( SDX_DATA_RESULT.SDX_DATA_TYPE_23_ID=SDX_DATA_TYPE_23.SDX_DATA_TYPE_23_ID )
AND ( SDX_DATA_TYPE_7.SDX_DATA_TYPE_7_ID=SDX_DATA_RESULT.SDX_DATA_TYPE_7_ID )
AND ( SDX_DATA_RUN.SDX_DATA_RUN_ID=SDX_DATA_RESULT.SDX_DATA_RUN_ID )
AND
SDX_DATA_RUN.RUN_START_TS BETWEEN '03-JAN-2019 01:00:00' AND '03-JAN-2020 01:00:00'
AND DW.SDX_DATA_RUN.TEST_TYPE IN (5, 11)
AND SDX_DATA_RUN.TEST_TYPE IN (5, 11)
